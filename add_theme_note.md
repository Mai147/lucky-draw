# Khi thêm 1 theme mới
1. Tạo file theme mới trong constants/Theme
2. Define vào constants/Theme/Theme.ts, thêm ThemeName
3. Thêm ThemeItem vào components/Modal/ThemeModal/index.tsx
4. Define variable trong index.css
5. Define variable trong tailwind.config.js
